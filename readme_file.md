# 📊 株最強分析くん

日本株の財務データを自動分析し、100点満点でスコアリングするStreamlitアプリケーション

## 🌟 主な機能

### 1. 財務分析スコアリング
- IRBANKから財務データを取得
- 9項目の財務指標を自動評価（100点満点）
- 視覚的な円グラフで総合スコアを表示

### 2. 株価チャート
- Yahoo Financeから株価データを取得
- マルチタイムフレーム対応（5分足〜月足）
- ローソク足チャートで表示

### 3. 履歴管理
- 分析履歴を自動保存
- 過去の分析結果と比較
- 月間ランキング表示

## 📋 評価項目

| 項目 | 基準 | 配点 |
|------|------|------|
| 経常収益 | 右肩上がり | 15点 |
| EPS | 右肩上がり | 15点 |
| 総資産 | 増加傾向 | 10点 |
| 営業CF | プラス＆増加 | 10点 |
| 現金等 | 積み上がり | 10点 |
| ROE | 7%以上 | 10点 |
| 自己資本比率 | 50%以上 | 10点 |
| 1株配当 | 非減配 | 10点 |
| 配当性向 | 40%以下 | 10点 |

**合計: 100点満点**

## 🚀 使い方

### ローカル環境で実行

1. **リポジトリをクローン**
```bash
git clone https://github.com/yourusername/stock-analyzer.git
cd stock-analyzer
```

2. **依存パッケージをインストール**
```bash
pip install -r requirements.txt
```

3. **アプリを起動**
```bash
streamlit run app.py
```

4. **ブラウザで開く**
```
http://localhost:8501
```

### Streamlit Cloudでデプロイ

1. このリポジトリをGitHubにプッシュ
2. [Streamlit Cloud](https://share.streamlit.io)にアクセス
3. GitHubアカウントで認証
4. 「New app」→ リポジトリを選択 → Deploy
